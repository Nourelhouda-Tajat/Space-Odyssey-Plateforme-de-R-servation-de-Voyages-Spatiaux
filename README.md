# Space Tourism Booking App

## Description
Application interactive de réservation de voyages spatiaux, avec formulaire dynamique, calcul de prix en temps réel et tickets imprimables.

## Fonctionnalités Clés
- Connexion simulée et gestion de session (`localStorage`)  
- Formulaire adaptatif selon destination et package  
- Calcul instantané du prix et des extras  
- Validation côté client (email, téléphone, dates, champs requis)  
- Création, édition et suppression de réservations  
- Génération de tickets imprimables avec QR code

## Technologies
HTML5, CSS3 (TailwindCSS), JavaScript (ES6+), `localStorage`, GitHub Pages

## Avancement
User Stories complétées : 1 à 6 et 10  

## Utilisation
1. Ouvrir `index.html` dans un navigateur  
2. Se connecter ou continuer en tant que "Guest"  
3. Créer et gérer vos réservations  
4. Visualiser et imprimer vos tickets
